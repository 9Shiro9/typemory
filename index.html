How to play · settings

Learn mode

Each word levels up as you master it: 📖 word shown → ⚡ flashes, then vanishes mid-air → 🧠 meaning only → ❓ surprise checks where you type the meaning.

Controls

Just type — the lowest zombie whose word starts with that letter gets targeted. No German keyboard needed: for ä ö ü ß type a o u s (ae oe ue ss work too) · Tab/💡 reveals a letter (costs points) · Esc/⏸ pause · Mistakes make the zombie faster. Let 5 through and you're bitten.

Install on iPhone 📲

1 · Tap the Share button ⬆ in Safari's toolbar
2 · Scroll down, tap "Add to Home Screen"
3 · Tap Add — Typemory becomes an app that works offline 🧟
